# Pocket Rocket Analyst

Pocket Rocket Analyst is a lightweight, CLI-based market analysis engine focused on
price structure, drawdowns, volatility, regime detection, and capital behaviour.

It is designed to run in constrained environments (Termux / Debian) with minimal
dependencies and reproducible outputs.

---

## Features

- OHLCV-based scoring engine
- Max drawdown & volatility analysis
- Regime detection (building / choppy / mixed)
- Deployment-quality & capital identity labels
- CSV + TXT reports
- CLI-first workflow

---

python3 pocket_analyst.py watchlists/watchlist.txt 10
